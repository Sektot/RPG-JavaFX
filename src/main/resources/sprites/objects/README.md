# Interactive Object Sprites

Place interactive object sprites in this folder.

## Required Sprites:

### Core Objects (sizes vary):
- `chest.png` - Treasure chest (48x48 pixels)
- `altar.png` - Shrine/altar (64x64 pixels)
- `fountain.png` - Healing fountain (64x64 pixels)
- `shop_table.png` - Shop counter (96x64 pixels)
- `campfire.png` - Rest area campfire (48x48 pixels)
- `portal.png` - Portal/exit (64x64 pixels)
- `statue.png` - Decorative statue (48x64 pixels)

## Sprite Requirements:
- Format: PNG (recommended), JPG, or GIF
- Size: Varies by object (see above)
- Transparent background recommended
- Should be recognizable when placed in room

## Naming Convention:
The sprite file name must match the object type in lowercase:
- ObjectType.CHEST → `chest.png`
- ObjectType.ALTAR → `altar.png`
- ObjectType.FOUNTAIN → `fountain.png`
- etc.

## Optional Variations:
- `chest_open.png` - Open chest variant
- `campfire_lit.png` - Lit campfire animation frames

## Free Sprite Resources:
- OpenGameArt.org
- itch.io (search "dungeon objects" or "pixel art props")
- Kenney.nl (free game assets)

# Player Sprites

Place player animation sprites in this folder.

## Required Sprites (32x32 pixels recommended):

### Walking Animations (4 frames each):
- `walk_down_0.png` to `walk_down_3.png` - Walking down
- `walk_up_0.png` to `walk_up_3.png` - Walking up
- `walk_left_0.png` to `walk_left_3.png` - Walking left
- `walk_right_0.png` to `walk_right_3.png` - Walking right

## Sprite Requirements:
- Format: PNG (recommended), JPG, or GIF
- Size: 32x32 pixels (can be adjusted)
- Transparent background recommended
- Frame naming: `animation_name_frame_number.png`

## Example Frame Sequence:
```
walk_down_0.png  -> Standing/first step
walk_down_1.png  -> Mid-step
walk_down_2.png  -> Standing/second step
walk_down_3.png  -> Mid-step
```

## Free Sprite Resources:
- OpenGameArt.org
- itch.io (search "top-down character sprites")
- Kenney.nl (free game assets)

# Enemy Sprites

Place enemy sprites in this folder.

## Required Sprites (32x32 to 64x64 pixels):

### Basic Enemies:
- `enemy_basic.png` - Default enemy sprite

### Optional Enemy Variants:
- `enemy_boss.png` - Boss enemy
- `enemy_elite.png` - Elite enemy
- `enemy_swarm.png` - Small swarm enemies
- `enemy_ranged.png` - Ranged enemy
- `enemy_tank.png` - Tank/heavy enemy

## Sprite Requirements:
- Format: PNG (recommended), JPG, or GIF
- Size: 32x32 to 64x64 pixels
- Transparent background recommended
- Can be animated (use frame naming: `enemy_basic_0.png`, `enemy_basic_1.png`, etc.)

## Tips:
- Enemies should face down by default
- Use red/dark colors to distinguish from player
- Larger sprites for boss enemies (64x64 or bigger)

## Free Sprite Resources:
- OpenGameArt.org
- itch.io (search "top-down enemy sprites")
- Kenney.nl (free game assets)
